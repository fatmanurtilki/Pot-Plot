# 🍳 Pot & Plot - Yemek Pişene Kadar Ders Çalış

React + Vite + Tailwind CSS ile geliştirilmiş, yemek pişirme sürelerine dayalı ders çalışma zamanlayıcısı.

## Özellikler
- 13 farklı yemek tenceresi, 13 farklı içecek fincanı
- Yemek/İçecek süresine göre geri sayım
- Mola modu
- Yemek ekleme/düzenleme/silme (CRUD)
- LocalStorage ile veri saklama
